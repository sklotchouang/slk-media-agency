Video Testimonials Folder
==========================

This folder is for video testimonial files that will be displayed in the horizontal sliding video testimonial section on the website.

Instructions:
1. Drop your video testimonial files here
2. Recommended format: MP4
3. Recommended naming convention:
   - testimonial-1.mp4
   - testimonial-2.mp4
   - testimonial-3.mp4
   etc.

4. The videos will be automatically referenced in the website's video testimonial carousel
5. Make sure videos are optimized for web (reasonable file size)
6. Recommended resolution: 1280x720 or 1920x1080

Note: Currently the website references placeholder paths. Update the HTML file to point to your actual video files once uploaded.
